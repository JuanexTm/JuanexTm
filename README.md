      <p>Chihiza is an immersive, interactive ritual that invites users to reconnect with the ancestral wisdom of Colombian Indigenous communities. Inspired by Muisca mythology and cosmology, the project simulates a sacred lake where users, acting as shamans, connect constellations to unlock ancestral knowledge in medicine, astronomy, and music. The experience blends procedural visuals, body interaction, and dynamic storytelling to raise awareness about the legacy and relevance of native traditions in modern society.

🧩 Responsibilities:

• Led the project planning, research, and development as Project Manager creating a Scrum-based workflow in Notion for team coordination.

• Conducted cultural research and mythology synthesis for experience design.

• Designed and implemented the entire procedural visual system in TouchDesigner.

• Built interaction terminals using TouchOSC for multi-user control.


🔍 Process Highlights:

• Physical immersive setup designed for live installation environments.

• Prototyped and iterated visual effects using real-time particle systems.

• In-depth cultural and mythological research.


🌱 Reviving ancestral wisdom through technology and interaction.</p>
